Introduction

The Scene class is the main entry point of this library. It is responsible for creating and managing a hierarchy of DisplayObjects, which are the visual elements that make up the scene.

It is coupled with a Viewport and a Grid, and allows for the following features:

  • Add, remove, and access DisplayObjects in the scene
  • Set the size and position of the Viewport
  • Set the size and spacing of cells in the Grid

The library possess models that can be added to the stage of the Scene. These models can be selected, dragged, resized and downloaded thanks to a few pluggable plugin managed with events.
