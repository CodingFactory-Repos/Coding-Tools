Class DragPlugin

<T: PixiObjecttype PixiObjecttypes/PixiObject>

extends PixiEventsclass PixiEventstypes/PixiEvents

A plugin that manages the dragging with a mouse or touch input of a given PixiObject.

Constructor

constructor(ref: Ttype parameter T)

Creates an instance of DragPlugin.

· ref - A reference to the PixiObject that the plugin will be applied to.

Properties

private readonly _element: Ttype parameter T[src]

The PixiObject that this plugin manages.

private _lastPosition: IPointData[src]

The last position of the cursor during a dragging operation.

private _startDrag = (event: FederatedPointerEvent) => void[src]

Handles the pointer down. When the pointer is pressed, register the position of the cursor local to the parent element. Add the event listener pointer move to the stage and the element to keep the dragging constant.

· event - - The pointerdown event.
private _stopDrag = (event: FederatedPointerEvent) => void[src]

Handles the pointer up. When the pointer is released, the isDragging property of the PixiObject is set to false and the event is removed. Update the element cursor to pointer.

· event - - The pointerup event.
private _moveDrag = (event: FederatedPointerEvent) => void[src]

Handles the pointer move. When the pointer move, the isDragging property of the PixiObject is set to false and the PixiObject position is recalculated. Update the element cursor to grabbing.

· event - - The pointermove event.
enableDragging = () => void[src]

Enables the dragging behavior for the associated PixiObject.

disableDragging = () => void[src]

Disables the dragging behavior for the associated PixiObject.
